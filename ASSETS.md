# Список изображений для приложения

Все картинки кладутся в `public/assets/`. Пока файла нет — приложение показывает
стильный плейсхолдер автоматически, ничего не ломается.

Рекомендуемые размеры: вертикальные/квадратные, ~800px по ширине, JPG, до 200 КБ.

## Архетипы (`public/assets/archetypes/`) — 4 шт, портрет/образ архетипа
- `ruler.jpg`, `lover.jpg`, `rebel.jpg`, `magician.jpg`

## Типы фигуры (`public/assets/body/`) — 4 шт, силуэты/примеры
- `slim-short.jpg`, `slim-tall.jpg`, `solid-short.jpg`, `solid-tall.jpg`

## Типы лица (`public/assets/face-type/`) — 3 шт
- `soft.jpg`, `balanced.jpg`, `hard.jpg`

## Формы лица (`public/assets/face-shape/`) — 6 шт, свои фото
- `square.jpg`, `oval.jpg`, `rectangle.jpg`, `round.jpg`, `diamond.jpg`, `triangle.jpg`
- Квадрат 800×800, лицо по центру (края подрежутся)
- В папке лежат запасные SVG-контуры — не мешают, можно удалить
